# Roadmap

## Phase 1: MVP (12 tuần)

**Mục tiêu:** Validate core value — "AI gợi ý món ăn hàng ngày có giá trị với user không?"

| Tuần | Deliverable |
|------|------------|
| W1-2 | Setup project, DB schema, recipe data import |
| W3-4 | Auth + Onboarding + Taste Profile |
| W5-6 | Recipe browsing + detail + search |
| W7-8 | AI Suggestion v1 (content-based + popularity) |
| W9-10 | Basic Meal Planning (tuần) |
| W11 | Nutrition display (calo, macro per recipe) |
| W12 | Testing, bug fix, soft launch (closed beta 100 users) |

**Scope:** Gợi ý bữa ăn + Recipe detail + Basic meal plan + Nutrition info + 1 dietary filter

## Phase 2: Learning & Growth (3 tháng)

- Collaborative filtering
- Full dietary filters
- Nutrition tracking dashboard + goals
- Smart Shopping List
- Push notifications
- Public launch (App Store + Google Play)

## Phase 3: Advanced AI & Monetization (3 tháng)

- NLP chatbot
- Pantry management
- Special occasion planning
- Voice assistant
- Freemium + Pro subscription
- Community features

## Phase 4: Scale & Optimize (Ongoing)

- A/B testing algorithms
- Multi-language
- Offline mode
- Wearable integration
- B2B extension
